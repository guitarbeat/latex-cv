## 2026-01-05 - [PDF Accessibility Metadata]
**Learning:** LaTeX documents often lack basic accessibility metadata (Title, Language) by default, which hinders screen readers. Using `\hypersetup` to explicitly set `pdftitle`, `pdflang`, etc., is a simple, high-impact fix.
**Action:** Always check `\hypersetup` configuration in LaTeX templates and ensure Title, Author, Language, and Subject are populated.
